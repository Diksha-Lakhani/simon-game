# Simon Game Project

A fun and interactive Simon Game built using HTML,CSS and JavaScript using JQuery. The game challenges players to memorize and repeat an increasingly long sequence of colors and sounds.

## Features

- Interactive gameplay with four colored buttons
- Random sequence generation
- Sound effects for each button
- Visual button animations
- Score/Level tracking
- Game Over notification with restart functionality
- Responsive design

## Technologies Used

- **HTML** – Structure of the game
- **CSS** – Styling
- **JavaScript** – Game logic
- **JQuery** - interactivity and animations

## How to Play

1. Press any key (or click the start button if implemented) to begin the game.
2. Watch the sequence of colors shown by the game.
3. Repeat the sequence by clicking the colored buttons in the correct order.
4. Each correct round adds a new color to the sequence.
5. The game ends when you click the wrong color.
6. Press any key again to restart and try to beat your previous level.
